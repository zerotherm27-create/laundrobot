How custom fields work in the chat flow

  • Steps 1–3 are the existing booking flow (welcome → category → service catalog)
  • Step 4 is the new custom fields step — each field is asked one at a time
  • Text fields → bot asks for typed reply
  • Select/dropdown fields → bot sends quick-reply chips so customer just taps
  • Optional fields show "(optional)" and customer can reply "skip"
  • All custom field answers appear in the Order Summary before confirmation
